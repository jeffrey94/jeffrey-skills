# Composition Element Schema

## Element Types

### Copy Elements

| Type | Description | Typical Position | Refinable |
|------|-------------|-----------------|-----------|
| H1 Headline | Primary attention text | top-center, center | text, size, color, position |
| Support Line | Secondary detail/proof | below headline | text, size, color |
| CTA Label | Action button text | bottom-center, center-right | text, color, size, shape |
| Trust Microcopy | Credibility markers | near CTA | text, visibility |
| Legal/Disclaimer | Regulatory fine print | bottom | text, size |

### Visual Elements

| Type | Description | Typical Position | Refinable |
|------|-------------|-----------------|-----------|
| Logo | Brand logo | top-left, top-right, bottom-center | position, size |
| Background | Scene/gradient/solid | full canvas | color, type, imagery |
| Product Image | Main product visual | center, center-right | swap, style |
| Photography | Hero/supporting photos | varies | swap, filter, crop |
| Icons/Badges | Supporting visuals | near CTA, corners | swap, style |

### Style Properties

| Property | Description | Refinable |
|----------|-------------|-----------|
| Color Palette | Overall color scheme | hex values, warmth/coolness |
| Typography | Font choices and sizing | family, weight, size ratio |
| Layout | Spatial arrangement | alignment, spacing, grid |
| Mood/Tone | Emotional feel | professional/playful/urgent |
| Contrast | Light/dark balance | ratio, readability |

## Preservation Defaults

When the user doesn't specify, preserve:
- Logo (position, size, colors)
- Brand color palette
- Overall aspect ratio
- Product/service visibility
- Regulatory disclaimers
